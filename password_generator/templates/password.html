Generate a Secure Password

This tool generates a password specific to the input text, and for any given input, it will always produce the same password. For example, instead of remembering a long password full of numbers and characters, you can simply remember, "my password is the lyrics to 'My Way' by Frank Sinatra." If you frequently forget your passwords, or you want one master password for your password manager, this is an easy option.

The complexity, stated simply, determines how long the password will be. A higher complexity will yield a shorter password. Specifically, the complexity has to do with how certain digits are subdivided in the generation algorithm.
